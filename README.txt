Welcome to the Astroweight Calculator!

This app allows you to calulate your weight on vaiour planets, stars and moons.

How To Load Project:
	- In the command line, type: "npm start".
	- In your web browser, navigate to "http://127.0.0.1:8080/".

How to test project: 
	- In the command line, type: "npm test".
